  1. Toggle Theme
     Click the moon/sun icon (top-right corner)

  2. Upload Resume
     → Navigate to "Resumes"
     → Drag a PDF/TXT/DOC/DOCX file to the upload area
     → Or click to browse
     → AI automatically parses it!

  3. Create Job
     → Navigate to "Jobs"
     → Fill the form (Title, Company, Skills, etc.)
     → Click "Create Job Posting"
     → Job is instantly created

  4. View Matches
     → Navigate to "Matches"
     → See AI-powered scores
     → Green = Great (80%+)
     → Yellow = Good (60-79%)
     → Blue = Possible (<60%)

┌──────────────────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

  Theme not switching?
    • Clear browser cache (Cmd+Shift+R or Ctrl+Shift+R)
    • Check browser console (F12)
    • Verify localStorage is enabled

  Upload not working?
    • Ensure backend is running (http://localhost:3000/health)
    • Check file type (PDF, TXT, DOC, DOCX only)
    • File must be under 10MB

  API errors?
    • Verify MongoDB is running
    • Check MISTRAL_API_KEY in backend/.env
    • Look at backend logs

  Styles not loading?
    • Hard refresh the page
    • Check that styles-modern.css exists
    • Verify server on port 8080
